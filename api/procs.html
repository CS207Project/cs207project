procs.corr module¶

This module implements the stored procedure that computes the correlation distance metric between a given TimeSeries argument and another TimeSeries

Notes

proc_main
the function called by the database on augmented selects
main
the function called by the database on triggers
procs.corr.main(pk, row, arg)¶

Compute the kernel correlation distance of a TimeSeries to a given reference TimeSeries

Parameters:
  • pk (string or int) – this is the primary key of the row of the database
  • row (dict) – dictionary consisting of column_name: value for this row in the database
  • arg (list) – TimeSeries in list form to which we will compute the kerndist
Returns:

[kerndist] of the TimeSeries in this row of the database to the arg
Return type:

list

Notes

Implementation of the kernel correlation distance is done in the _corr module
